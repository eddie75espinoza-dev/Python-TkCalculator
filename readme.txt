Python 3.8, librerías TkInter y Math

Calculadora (GUI) realizada con librerías Tkinter y math.

Tkinter es un binding (es una adaptación de una biblioteca para ser usada en un lenguaje de programación distinto de aquel en el que ha sido escrita) de la biblioteca gráfica Tcl/Tk para el lenguaje de programación Python. Se considera un estándar para la interfaz gráfica de usuario (GUI) para Python y es el que viene por defecto con la instalación para Microsoft Windows.

Esta aplicación muestra una interfaz ligera, agradable y fácil de usar por cuanto se trata de una calculadora básica.
